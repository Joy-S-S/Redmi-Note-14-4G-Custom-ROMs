ADB Debloat Guide

1

Enable Developer Options

Go to Settings → About Phone and tap "OS Version" 7 times until you see "You are now a developer".

2

Enable USB Debugging

Go to Settings → Additional Settings → Developer Options and enable "USB Debugging".

3

Connect to PC

Connect your device to PC via USB cable. When prompted, allow USB debugging on your device.

4

Run ADB Commands

Open command prompt/terminal and run the debloat commands from the sections above.

Example Command 
adb shell pm uninstall --user 0 com.miui.notes
